# 🎓 College Result Management System

A web-based application developed to simplify the management of student academic records. The system allows administrators to manage students, faculty, subjects, classes, and examination results while enabling students to securely access their results online.

---

## 📌 Features

- Admin Login
- Student Login
- Add/Edit/Delete Students
- Manage Classes
- Manage Subjects
- Add Examination Results
- View Student Results
- Performance Dashboard
- Student Progress Reports
- Responsive User Interface
